## 🎯 About the Project

The core idea behind this portfolio is to go beyond a simple resume. The goal was to create an interactive experience that reflects my passion for **Storytelling** and **Game Design**, featuring a native "Dark Mode" interface, *Liquid Glass* effects, and a responsive Bento Grid architecture.

### ✨ Key Features

- **Internationalization (i18n):** Native support for English (EN) and Portuguese (PT-BR), with automatic browser language detection.
- **Extreme Responsive Design:** Mobile-first approach with specific handling for iOS safe areas (Notch and Dynamic Island).
- **Fluid Animations:** Built with Framer Motion for smooth scroll-triggered transitions and micro-interactions.
- **Performance-Driven:** Optimized image rendering with lazy loading, asynchronous decoding, and SVG assets.

## 💻 Running Locally

If you want to explore the source code and run the portfolio on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/K0W4/kowa-portfolio.git](https://github.com/K0W4/kowa-portfolio.git)
   ```

2. **Navigate to the project folder:**
   ```bash
   cd kowa-portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

> The application will be running at `http://localhost:5173`.
